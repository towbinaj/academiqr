# Snake Game

A classic Snake game with customizable snake colors, insect collectibles, and dangerous bombs!

## Features

- **Customizable Snake Colors**: Choose from 5 different colors for your snake
- **Insect Collectibles**: Collect different insects for varying point values
  - 🐜 Ant: 1 point
  - 🐝 Bee: 3 points
  - 🦗 Cricket: 5 points
- **Bombs**: Avoid bombs or your snake will explode!
- **Responsive Design**: Works on different screen sizes
- **High Score Tracking**: Your highest score is saved locally

## How to Play

1. Open `index.html` in your web browser
2. Choose your preferred snake color
3. Click "Start Game"
4. Use the arrow keys to control the snake:
   - ↑: Move Up
   - ↓: Move Down
   - ←: Move Left
   - →: Move Right
5. Press the spacebar to pause/resume the game
6. Collect insects to grow your snake and earn points
7. Avoid bombs, walls, and your own tail!

## Game Rules

- The snake moves continuously in the direction you choose
- The snake grows longer when it eats an insect
- The game ends if the snake:
  - Hits a wall
  - Hits its own body
  - Hits a bomb (which causes an explosion)
- More bombs appear as your score increases
- Different insects are worth different point values

## Installation

No installation required! Simply download the files and open `index.html` in any modern web browser.

```
git clone https://github.com/yourusername/snake-game.git
cd snake-game
```

Then open `index.html` in your browser.
